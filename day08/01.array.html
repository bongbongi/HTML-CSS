자바스크립트의 배열을 유동 배열 : 크기와 데이터의 변경이 가능하다.

  • const myArray = ['a','b','c',1,2,3];
  • 데이터의 타입이 다르더라도 허용하고 있음
  • myArray = []; 초기값이 없는 빈 배열을 생성 후 데이터를 추가하여 크기 변경이 가능하다.
  • 각 데이터들은 컴마(,)로 분리되고, 자리는 index로 관리, 0부터 시작한다.
  • 길이(length) 속성을 사용하여 측정 크기가 변경될 때마다 자동으로 체크
  • index번호 순서의 뛰어넘음 등의 변칙적인 사용이 가능하다. 모두 undefined로 초기화 된다.
